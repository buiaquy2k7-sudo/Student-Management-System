# Student Management System - C++ OOP



## Yêu cầu đã đáp ứng
- Minh họa đóng gói
- Minh họa kế thừa
- Minh chứng đa hình
- Thể hiện tư duy trừu tượng
- Có 16 lớp
- Giao diện Console
- Mỗi lớp được tách thành file `.h` và `.cpp`

## Cách chạy

```bash
g++ main.cpp src/*.cpp -Iinclude -o app.exe
.\app.exe
```
